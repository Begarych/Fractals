# Fractals
A simple program using a module "math" that draws fractals and displays the image of the fractal on the screen
Work in progress...
Quick walkthrough:
defining mandelbrot:
        x = px / W * 3 - 2  
        y = py / H * 2 - 1  
        c = x + 1j * y
        z = 0j
